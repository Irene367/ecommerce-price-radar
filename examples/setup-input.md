# 示例输入：竞品监控表搭建

我是一人店铺，卖的是女士通勤托特包，主要平台是淘宝。

请帮我搭建一张竞品监控表，用来每天记录 8-12 个竞品的价格、优惠券、活动、库存、发货承诺、评价和卖点变化。

我的情况：

- 核心 SKU：大容量通勤托特包
- 本店标价：169 元
- 常用券后价：139 元
- 成本：78 元
- 最低希望毛利率：30%
- 主要竞争对手：同价位女包店、低价爆款店、品牌旗舰店
- 我能提供的数据：人工查看后的商品标题、到手价、优惠券、库存状态、发货承诺、评价数、评分、主图卖点和活动标签

请输出：

1. 竞品分层方式。
2. 每日监控表字段。
3. 哪些字段必须每天更新。
4. 缺失数据时如何标注置信度。
5. 首周执行建议。

注意：不要自动登录、不要爬取平台、不要自动改价，只基于我提供的数据设计监控方式。
